# express-mongodb-chat
Express, MongoDB を使ってシンプルなチャットアプリを開発。

## 機能
パスワード認証 / サインアップ / 記事の投稿・一覧表示 / 画像アップロード

## 使用技術
Express / MongoDB / Passport / Pug / Less


## ローカルでの動作方法
ローカルの MongoDB 起動
```
npm run db
```

ローカルサーバー起動
```
npm run serve
```

Less のコンパイル
```
lessc less/app.less css/app.css
```
